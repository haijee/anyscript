# anyscript

前端常用javascript函数工具库

# usage

## Installation

```sh
npm install anyscript

```

Or

```sh
yarn add anyscript

```

# Quick start

```js

import { test } from 'anyscript'

test() // 传入参数（属性）

```

# API

### 防抖函数 

debounce

| 属性 | 说明 | 类型| 默认值 | 其他 |
| --- | --- | --- | --- | --- |
| fn | 防抖的函数  | function | - |
| delay | 时间  | number | 300 |  

### 节流函数 

throttle

| 属性 | 说明 | 类型| 默认值 | 其他 |
| --- | --- | --- | --- | --- |
| fn | 防抖的函数  | function | - |
| delay | 时间  | number | 300 |
